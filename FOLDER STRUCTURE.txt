Description:
- The /app folder contains all Next.js routes (pages and layouts), including authentication, dashboard, admin, and profile.
- The /components folder contains all React UI components used throughout the app.
- The /lib folder contains utility/helper files and the Supabase client.
- The /public folder is for images and static assets.
- The /styles folder contains global CSS styles (used by Tailwind).
- Core config files for Tailwind, PostCSS, TypeScript, and environment variables are at the root.
- README.md provides project info and instructions.
- .env.local.example is for environment variable templates.
